- **npm** â‰¥ 9.0.0 or **yarn** â‰¥ 1.22.0 or **bun** â‰¥ 1.0.0

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/YashGupta404/cinema-reel-portfolio.git
   cd cinema-reel-portfolio
   ```

2. **Install dependencies**
   ```bash
   # Using npm
   npm install

   # Using yarn
   yarn install

   # Using bun
   bun install
   ```

3. **Start the development server**
   ```bash
   # Using npm
   npm run dev

   # Using yarn
   yarn dev

   # Using bun
   bun dev
   ```

4. **Open your browser**
   ```
   http://localhost:8080
   ```

### Build for Production

```bash

npm run build

# Preview production build locally

```

## ğŸ§© Components

### Cinema Components

#### `CurtainOpening`
The entrance animation featuring realistic red velvet theater curtains.

```tsx
<CurtainOpening onComplete={() => setShowContent(true)} />
```

**Features:**
- 7-fold velvet curtain with GSAP animations
- Gold fringe with realistic tassel details
- Top valance with decorative gold trim
- "Enter Theater" button with vintage styling

#### `FilmCountdown`
Classic movie countdown from 5 to 1.

```tsx
<FilmCountdown onComplete={handleStart} />
```

#### `ProjectorModel`
3D projector with volumetric light using Three.js.

```tsx
<Canvas>
  <ProjectorModel />
</Canvas>
```

### Scene Components

#### `AboutScene`
Movie trailer-style introduction with silent film aesthetics.
- Animated phrase sequence ("In a world...", etc.)
- Vintage decorative ornaments
- Statistics display

#### `SkillsScene`
Film strip format displaying technical skills.
- Horizontal scrolling film frames
- Category-based skill groupings
- Progress bar indicators

#### `ProjectsScene`
Movie poster grid showcasing projects.
- Hover effects with flip animation
- Genre tags and ratings
- Director credits

#### `ContactScene`
Theater seating arrangement for contact options.
- Interactive seat selection
- Email, LinkedIn, GitHub, Twitter links
- Contact form modal

## ğŸ¨ Customization

### Theme Configuration

The theme is configured in `tailwind.config.ts` with cinema-specific colors:

```typescript
theme: {
  extend: {
    colors: {
      velvet: {
        DEFAULT: "hsl(var(--velvet))",
        dark: "hsl(var(--velvet-dark))",
        light: "hsl(var(--velvet-light))",
      },
      projector: "hsl(var(--projector-light))",
      film: {
        sepia: "hsl(var(--film-sepia))",
        cream: "hsl(var(--film-cream))",
      },
      theater: {
        gold: "hsl(var(--theater-gold))",
      },
    },
    fontFamily: {
      cinema: ['Playfair Display', 'serif'],
      film: ['Cormorant Garamond', 'serif'],
    },
  },
}
```

Pre-built cinema animations available:

| Animation | Duration | Description |
